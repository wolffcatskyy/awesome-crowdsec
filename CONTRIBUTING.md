# Contributing to Awesome CrowdSec

Thanks for your interest in contributing!

## Guidelines

- Search existing entries before adding
- Use the format: `[Name](link) - Description`
- Keep descriptions concise (under 100 characters)
- Add new entries in alphabetical order within their section
- Ensure the project is actively maintained
- Projects should be related to CrowdSec

## Pull Request Process

1. Fork the repository
2. Add your entry following the format above
3. Submit a PR with a clear description

## Quality Standards

- Repository should have a README
- Project should be functional (not abandoned)
- Prefer projects with recent commits
