# Roadmap

This document outlines the planned development and growth trajectory for Awesome CrowdSec.

## Current State

The list currently includes:
- **6** Official Resources
- **15** Bouncers (firewall, web servers, cloud, CMS)
- **2** Firewall Integrations (OPNsense, pfSense)
- **3** Web UIs & Management tools
- **2** Blocklist & Threat Intelligence tools
- **8** Docker Stacks & Examples
- **1** Dashboard solution
- **4** Platform Integrations

Total: **41 curated resources** across 10 categories.

## Short-term Goals (Q1-Q2 2026)

### Expand Coverage

- [ ] Add more container orchestration bouncers (Podman, LXC)
- [ ] Include cloud-native integrations (AWS WAF, Azure Firewall, GCP)
- [ ] Add more CMS bouncers (Drupal, Joomla, Magento)
- [ ] Include additional reverse proxy solutions (Envoy, Kong, Ambassador)

### Improve Discoverability

- [ ] Add badges for project activity (last commit, stars)
- [ ] Include brief "why use this" notes for standout projects
- [ ] Add a "Getting Started" section for newcomers

### Quality Assurance

- [ ] Audit all links for validity (automated check)
- [ ] Review projects for maintenance status
- [ ] Remove abandoned projects (no commits in 2+ years)

## Mid-term Goals (Q3-Q4 2026)

### Community Growth

- [ ] Submit to the official [awesome](https://github.com/sindresorhus/awesome) list
- [ ] Engage with CrowdSec community for submissions
- [ ] Create issue templates for suggesting new resources
- [ ] Add a "New This Month" section for visibility

### New Categories

- [ ] **Tutorials & Guides** - Written and video tutorials
- [ ] **API Clients** - SDKs and client libraries
- [ ] **Log Parsers** - Custom parsers for various applications
- [ ] **Scenarios** - Community-contributed detection scenarios
- [ ] **Managed Services** - SaaS offerings built on CrowdSec

### Automation

- [ ] Add link checker GitHub Action
- [ ] Implement star count badges via shields.io
- [ ] Add automated "last updated" tracking

## Long-term Goals (2027+)

### Ecosystem Maturity

- [ ] Become the definitive CrowdSec resource directory
- [ ] Partner with CrowdSec team for official endorsement
- [ ] Add translations for non-English communities
- [ ] Create a companion website with search and filtering

### Content Depth

- [ ] Add comparison tables for similar tools
- [ ] Include performance benchmarks where available
- [ ] Document integration patterns and best practices
